# Banking Service Simulator

A simple C# console application that simulates a basic banking service system.
The program allows users to perform common banking operations through a terminal interface.

## Features

* Lock and unlock a bank card
* Check account balance
* Make money transfers
* View transaction history (account statement)

## Technologies

* C#
* .NET
* Console Application

## Project Purpose

This project was created to practice fundamental C# concepts such as:

* Variables and data types
* Conditional statements
* Loops
* Lists and basic data structures
* User input and console interaction

## Example

```
1 - Check Balance
2 - Transfer Money
3 - Lock Card
4 - Unlock Card
5 - View Transaction History
```

The user selects an option and the system performs the corresponding banking operation.
